web开发学习——W3C

web开发
  • W3C
  • web


Web 的规则制定主体是 W3C；W3C 指的是万维网联盟（World Wide Web Consortium）；W3C 将各种规范订立为 Web 标准；最核心的 Web 标准是 HTML、CSS、XML。

一、简介：

W3C指万维网联盟（World Wide Web Consortium）创建于1994年10月，这个会员组织的工作是对 Web 进行标准化，创建并维护 WWW 标准。W3C 最重要的工作是发展 Web 规范（称为推荐，Recommendations），这些规范描述了 Web 的通信协议（比如 HTML 和 XHTML）和其他的构建模块。

二、W3C程序

W3C的标准化程序分为7个步骤，在 W3C 发布某个新标准的过程中，规范是通过下面的严格程序由一个简单的理念逐步确立为推荐标准的：

  1. W3C 收到一份提交
  2. 由 W3C 发布一份记录 由 W3C
  3. 创建一个工作组
  4. 由 W3C 发布一份工作草案
  5. 由 W3C 发布一份候选的推荐
  6. 由 W3C 发布一份被提议的推荐
  7. 由 W3C 发布推荐
